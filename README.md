# caribbeanciviclab.com

Single-page, intentionally work-in-progress website for Caribbean Civic Lab (CCL). It exists to show intent, gather feedback, and provide a stable link for grant applications.

## Local preview

Open `index.html` in a browser.

## GitHub Pages setup (custom domain)

Repo: `CaribbeanCivicLab/caribbeanciviclab.com`

In GitHub: Settings -> Pages

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

Custom domain:

- Set Custom domain to `caribbeanciviclab.com`
- Ensure `Enforce HTTPS` is enabled once DNS is correct

DNS (apex domain) A records for GitHub Pages:

- `185.199.108.153`
- `185.199.109.153`
- `185.199.110.153`
- `185.199.111.153`

This repo includes:

- `CNAME` (GitHub Pages custom domain)
- `.nojekyll`
