Yuh Hear Dem?

https://yuhheardem.com/

Yuh Hear Dem? is an AI-powered transparency tool that makes Barbados Parliament debates searchable and easier to follow. You can ask questions and get "receipts": timecoded source links back to the exact moment something was said.

Why this matters

Small states are often forced into expensive, vendor-led digital transformation. AI lowers the cost of building useful civic infrastructure if we keep it lightweight, source-cited, and locally maintainable.

Nonpartisan posture

Indexing + access only. No politician scoring. No "truth meter." Everything points back to primary sources.

90-Day Intent (Barbados-First)

This is the near-term plan we want feedback on. It is deliberately small and deliverable.

Deployment hardening
Reliability, UX improvements, an operations runbook, and clearer source citations.
Replication kit
Parameterized ingest + transcript pipeline + docs so another island is a configuration exercise, not a rewrite.
Talent pipeline
A small local developer cohort contributing to a real public system (fellowship / hack-nights).

Note: regional replication is part of the long-term story, but the focus here is a Barbados-first flagship that can be exported.

What We Need

We are looking for a small number of aligned partners to make this real without building a heavy institution too early.

Fiscal sponsor
An organization that can host the initiative and receive/hold restricted funds for pilot delivery.
Seed underwriting
Catalytic funding for servers/compute, dev stipends, and Barbados-first delivery (pilot scale).
Partners + feedback
Journalists, civil society, academia, and technologists who can pressure-test usefulness and neutrality.
